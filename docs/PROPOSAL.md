## 2. 背景：なぜ今、AI検索対策が必要か

### 2.1 市場の変化

- **AI検索からのトラフィックが前年比527%増加**（AllAboutAI 2025）
- ChatGPT・Perplexity・Gemini の月間ユーザーは合計で数億人規模
- Google検索のAI Overview表示率はデスクトップキーワードの約9.5%（米国では16%）

### 2.2 従来のSEOだけでは不十分な理由

| 課題 | 説明 |
|------|------|
| クリック率の低下 | AI Overview表示時にクリック率が約34.5%減少 |
| 可視性の不透明さ | AI検索で自社がどう表示されているか、従来ツールでは計測できない |
| 対策の曖昧さ | E-E-A-T・構造化データ・コンテンツ… 何を優先すべきか判断が難しい |

### 2.3 貴社に当てはまる課題はありませんか？

- □ AI検索で自社がどう表示されているか分からない
- □ 競合は出てくるのに自社は出てこない
- □ AIプラットフォームごとに手動確認するしかない
- □ 「AI対策」の効果測定ができない
- □ 何から手をつければいいのか分からない

## 3. AIO Insight が提供する価値

### 3.1 3つの価値提案

| 価値 | 内容 |
|------|------|
| **可視化** | AI検索での現状を6カテゴリでスコアリング。数値で「見え方」を把握 |
| **具体化** | 課題ごとに改善コード（JSON-LD・metaタグ）をAIが自動生成。コピペで実装可能 |
| **継続化** | 月次モニタリングでスコア変動・競合動向を追跡。効果測定の根拠に |

### 3.2 設計原則

> **「何が問題か」は無料で見せる（診断）**  
> **「どう直すか」は有料で提供する（改善コード）**

全てAIが自動生成。人的介在は最小限で、貴社の工数負担を抑えます。
