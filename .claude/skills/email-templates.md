# 営業メール テンプレート・トーンルール — アイデンティティ型スキル

このスキルは、FormPilot が自動送信する営業メールのトーン・構成・文体を定義する。メールテンプレートの生成、A/Bテストの文面作成、フォローアップメールの作成時に必ず参照する。

## インプット
- リード情報（企業名、業種、LLMO診断スコア、担当者名（判明時のみ））
- メールの種類（初回アプローチ / フォローアップ / 診断結果送付 / 商談誘導）

## アウトプット
- 件名 + 本文のメールテンプレート
- 本スキルのトーンルールに100%準拠したメール文面

---

## トーンの基本原則

### ペルソナ: 受信者像
- 日本の中小企業経営者（従業員10-100名）
- 年齢40-60代が多い
- ITリテラシーは中程度（Webサイトは持っているが、AI検索は詳しくない）
- 毎日大量のメールを受信しており、営業メールへの警戒心が高い
- 「うちには関係ない」「また営業か」が最初の反応

### トーンマトリクス

| 要素 | レベル | 具体的な表現 |
|------|--------|------------|
| 敬語レベル | 丁寧語ベース | 「です・ます」調。「ございます」は1通に1回まで（過剰に丁寧だと営業臭が増す） |
| 距離感 | やや近い専門家 | 「売り込み」ではなく「情報提供」の姿勢。コンサルタントが知見を共有するトーン |
| 緊急度 | 静かな危機感 | 煽りすぎない。データに語らせる。「大変です！」ではなく「ご存知でしょうか」 |
| 専門性 | 噛み砕いて伝える | 専門用語は使ってよいが、必ず（ ）で補足する |
| 長さ | 短い | 本文は200字以内を目標。スクロールなしで読める長さ |

### 使ってよい表現
- 「ご存知でしょうか」（問いかけ型。押し付けない）
- 「〇〇業界では」（業種を特定して自分ごと化）
- 「無料で」「30秒で」（行動のハードルを下げる）
- 「お忙しいところ恐れ入ります」（冒頭の定型として1回だけ）
- 「ご参考までに」（押し付けない姿勢）

### 使ってはいけない表現
- 「今すぐ」「急いで」「大至急」（煽り表現。警戒心を高める）
- 「他社様も導入されています」（具体的な数字なしの社会的証明は逆効果）
- 「お得なキャンペーン」「特別価格」（安売り感。ブランドを毀損する）
- 「ChatGPTで試してみてください」（相手がやらない前提で書く。やらせるのではなく結果を見せる）
- 「弊社は」で始まる文（自分語りから始めない。相手の課題から始める）
- 絵文字・顔文字（ビジネスメールには不適切）

## メールテンプレート

### 1. 初回アプローチメール

**目的**: LLMO未対策企業に対して、AI検索での可視性の課題を伝え、無料診断への誘導を行う

**件名パターン**（A/Bテスト用に複数用意）:
- A: 「[企業名]様のAI検索での表示状況について」
- B: 「[業種]業界のAI検索対策、ご存知ですか」
- C: 「ChatGPTで[業種]を検索すると、[企業名]様は…」

**本文構成**:
```
[1行目] 挨拶（1文のみ）
[2-3行目] 課題の提示（データベース）
[4-5行目] 具体的な発見事項（この企業に固有の内容）
[6行目] 解決策の概要（1文）
[7行目] CTA（無料診断への誘導）
[8行目] 署名
```

**テンプレート例**:
```
[企業名] [担当者名（不明の場合は「ご担当者」）]様

お忙しいところ失礼いたします。
AI検索最適化サービス「AIO Insight」の[署名者名]と申します。

現在、ChatGPTやPerplexityなどのAI検索を利用する方が急増しており、
[業種]業界でも「AIに聞いて業者を探す」ケースが増えています。

[企業名]様のWebサイトを拝見したところ、AI検索での可視性に
改善の余地がある可能性がございます。

30秒の無料診断で現状を確認いただけます。
▼ 無料AI可視性診断
[診断URL]

ご参考になれば幸いです。

---
[署名]
```

**文字数目安**: 件名30字以内、本文150-200字

### 2. フォローアップメール（初回から3日後）

**目的**: 初回メール未開封・未反応のリードに対する再アプローチ

**件名パターン**:
- A: 「先日のAI検索に関するご案内の件」
- B: 「[業種]業界の最新AI検索データをお送りします」

**本文構成**:
```
[1行目] 前回メールへの言及（1文）
[2-3行目] 追加の価値提供（新しい情報やデータ）
[4行目] CTA（前回と同じか、別の切り口）
[5行目] 署名
```

**テンプレート例**:
```
[企業名] [担当者名]様

先日、AI検索での可視性についてご案内いたしました。

参考情報として、[業種]業界では競合他社の約[X]%が
すでにAI検索対策を開始しているというデータがございます。

対策状況の無料診断は30秒で完了いたします。
▼ 無料AI可視性診断
[診断URL]

---
[署名]
```

**文字数目安**: 本文100-150字（初回より短く）

### 3. 診断結果送付メール

**目的**: 無料診断を実施した企業に結果を送付し、有料プランへの転換を促す

**件名**: 「[企業名]様のAI可視性診断レポートが完成しました」

**本文構成**:
```
[1行目] 診断実施の御礼
[2-3行目] 診断結果サマリー（スコアとランク）
[4-5行目] 主要な改善ポイント（最大3つ）
[6行目] 有料プランの簡潔な紹介
[7行目] CTA（詳細レポート閲覧 or プラン申込）
[8行目] 署名
```

**テンプレート例**:
```
[企業名] [担当者名]様

AI可視性診断にお申し込みいただき、ありがとうございます。
診断レポートが完成しましたのでお知らせいたします。

■ 診断結果
  総合スコア: [X]点 / 100点（ランク: [ランク]）

■ 主な改善ポイント
  1. [改善ポイント1]
  2. [改善ポイント2]
  3. [改善ポイント3]

詳細レポートと具体的な改善プランはこちらからご確認いただけます。
▼ 診断レポートを見る
[レポートURL]

---
[署名]
```

### 4. 商談誘導メール

**目的**: 診断結果を確認済みのリードに対して、有料プランの契約を促す

**件名**: 「[企業名]様向けのAI検索対策プランのご提案」

**本文構成**:
```
[1行目] 診断レポート閲覧の御礼
[2-3行目] スコアの改善余地（ポジティブな表現）
[4行目] 推奨プランの紹介（1文）
[5行目] 料金と開始までの手順（簡潔に）
[6行目] CTA（申込 or 相談予約）
[7行目] 署名
```

## A/Bテスト設計ルール

### テストすべき変数（優先順位順）
1. **件名**（開封率への影響が最大）
2. **CTA文言**（クリック率への影響）
3. **本文の長さ**（短い vs やや詳しい）
4. **業種特化度**（汎用文面 vs 業種固有のデータ引用）

### テストしてはいけないこと
- トーンの根本的変更（煽り系 vs 控えめ系のテストは不要。控えめ系で統一）
- 1通のメールで2つ以上の変数を同時にテスト
- サンプルサイズ50未満での判定

### 判定基準
- 開封率: 25%以上を合格ライン
- クリック率（CTA）: 5%以上を合格ライン
- 統計的有意差: 最低でもp < 0.05 で判定

```

AIO Insight — AI検索最適化サービス
フルフィル株式会社
https://aio-rouge.vercel.app
━━━━━━━━━━━━━━━━━━━━
```

署名は最小限に留める。電話番号・住所は入れない（ゼロミーティング営業の方針に沿い、問い合わせはWebフォーム経由に誘導する）。
