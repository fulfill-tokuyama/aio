# AIO / LLMO 専門知識 — 辞書型スキル

このスキルは、AI検索最適化（AIO）の専門知識をエージェントに注入するためのものである。AIO Insight のスキャンロジック、診断レポートの生成、LP のコピーライティング、顧客への提案内容の設計において、このスキルを参照して判断する。

## インプット
- 企業名、業種、WebサイトURL
- または、AIO関連の意思決定が必要な場面

## アウトプット
- LLMO診断結果（スコア + 改善提案）
- 顧客向けの説明テキスト
- スキャンロジックの判断基準

## 用語辞書

| 用語 | 正式名称 | 定義 | 顧客向け説明（平易な日本語） |
|------|---------|------|--------------------------|
| LLMO | Large Language Model Optimization | 大規模言語モデルの出力においてブランドや企業が言及されやすくなるための最適化手法 | AIに聞かれたとき、あなたの会社が「おすすめ」として出てくるようにする対策 |
| AIO | AI Optimization | LLMO を含むAI検索全般への最適化の総称。AIO Insight ではこちらを顧客向けブランド用語として使用 | AI検索であなたのビジネスが見つかるようにする対策 |
| AEO | Answer Engine Optimization | 回答エンジン（AI検索、音声アシスタント等）向けの最適化 | AIやSiriに質問されたとき、回答に含まれるようにする対策 |
| GEO | Generative Engine Optimization | 生成AIエンジン向けのコンテンツ最適化。学術研究で使われる用語 | （顧客向けにはAIOで統一する） |
| E-E-A-T | Experience, Expertise, Authoritativeness, Trustworthiness | Googleの品質評価基準。経験・専門性・権威性・信頼性 | あなたのサイトが「信頼できる情報源」としてAIに認識されるための4つの条件 |
| 構造化データ | Structured Data / Schema.org | Webページの情報を機械が理解しやすい形式で記述したもの | AIがあなたのサイトの内容を正しく理解するための「読み取りガイド」 |
| AI可視性 | AI Visibility | AI検索においてブランドが言及される度合い | AIに質問したとき、あなたの会社が出てくる確率 |
| AI検索 | AI Search | ChatGPT、Perplexity、Gemini等のAIを使った情報検索 | ChatGPTやGoogleのAI機能で調べること |

## 市場データ（顧客説得用）

### AI検索の普及状況
- ChatGPT: 月間アクティブユーザー 5億人超（2025年時点）
- Perplexity: 月間1500万人以上、前年比400%超の成長
- Google AI Overview: 米国検索結果の約20-30%に表示
- 日本国内: 経営者のAIツール利用率は年々増加。特に情報収集・市場調査での利用が急増

### 検索行動の変化
- 従来: Google検索 → 上位10サイトを比較 → 問い合わせ
- 現在: AI検索 → AIが推薦する2-3社を確認 → 問い合わせ
- AI検索ではSEOの上位表示だけでは不十分。AIの学習データに含まれ、かつAIが「推薦に値する」と判断する情報設計が必要

### 顧客に刺さるフレーズ
- 「AIに聞いても、あなたの会社は出てきません」
- 「競合はすでにAI検索で推薦されています」
- 「SEO対策だけでは、もう見つけてもらえない時代です」
- 「AI検索で言及されない = 存在しないのと同じ」
- 「対策している企業はまだ1%未満。今が先行者利益を獲得するチャンスです」

## 診断ロジック（スキャンエンジン用）

### LLMO対策状況の判定基準

企業のWebサイトを以下の観点でスキャンし、0-100点のスコアを算出する。

| 評価項目 | 配点 | 判定方法 |
|---------|------|---------|
| 構造化データの実装 | 20点 | JSON-LD / Schema.org マークアップの有無と網羅性 |
| E-E-A-T シグナル | 20点 | 著者情報、会社概要、実績・受賞歴、資格の明示 |
| FAQ / Q&A コンテンツ | 15点 | よくある質問ページの有無、質と量 |
| コンテンツの専門性 | 15点 | 業界固有の専門コンテンツの深さ、独自データの有無 |
| 技術的SEO基盤 | 10点 | サイトスピード、モバイル対応、HTTPS、サイトマップ |
| 外部からの言及・引用 | 10点 | 他サイトからの被リンク、メディア掲載、口コミ |
| AI検索での言及テスト | 10点 | 実際にChatGPT/Geminiで業界キーワードを検索し、言及されるか |

### スコア帯と診断メッセージ

| スコア | ランク | 診断メッセージ（顧客向け） |
|--------|--------|--------------------------|
| 80-100 | A（優良） | AI検索でも十分に可視化されています。さらなる最適化で圧倒的な優位を確立しましょう |
| 60-79 | B（良好） | 基本的な対策はできていますが、競合に差をつけるには追加の最適化が必要です |
| 40-59 | C（要改善） | AI検索での可視性に課題があります。今すぐ対策を始めることをお勧めします |
| 20-39 | D（危険） | AI検索でほぼ見つかりません。競合に大きく遅れを取っている状態です |
| 0-19 | E（未対策） | AI検索への対策がほぼゼロの状態です。緊急に対策が必要です |

### リードスコアリング（営業優先度の判定）

FormPilot のパイプラインでリードの営業優先度を判定する基準。

| 判定軸 | 高優先度の条件 | 理由 |
|--------|--------------|------|
| 業種 | ブライダル、美容医療、不動産、士業、飲食 | 検索依存度が高く、AIO対策のROIが大きい |
| 企業規模 | 従業員10-100名 | IT専任者がおらず外部サービスへのニーズが高い |
| LLMO対策状況 | スコアD-E（0-39点） | 改善余地が大きく、効果を実感しやすい |
| Web投資状況 | SEO対策済み or Web広告出稿中 | デジタルマーケへの投資意欲がある証拠 |
| 地域 | 首都圏・関西・福岡 | 競争が激しく、差別化ニーズが高い |
